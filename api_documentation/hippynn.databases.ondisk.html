Parameters:

  • directory ‚Äì directory path where the files are stored

  • name ‚Äì prefix for the arrays.

This function loads arrays of the format f‚Äù{name}{db_name}.npy‚Äù for each variable db_name in inputs and targets.

Note

This database loader does not support the allow_unfound setting in the base Database. The variables to load must be set explicitly in the inputs and targets.
