Parameters:

  • arr_dict – dictionary mapping strings to numpy arrays

  • inputs – list of strings for input db_names

  • targets – list of strings for output db_namees

  • seed – int, for random splitting

  • test_size – fraction of data to use in test split

  • valid_size – fraction of data to use in train split

  • num_workers – passed to pytorch dataloaders

  • pin_memory – passed to pytorch dataloaders

  • allow_unfound – If true, skip checking if the needed inputs and targets are found. This allows setting inputs=None and/or targets=None.

  • quiet – If True, print little or nothing while loading.
