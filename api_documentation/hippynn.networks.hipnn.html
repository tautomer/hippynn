Parameters

  • n_features ‚Äì width of each layer

  • n_sensitivities ‚Äì number of sensitivity functions

  • dist_soft_min ‚Äì midpoint of first sensitivity function

  • dist_soft_max ‚Äì midpoint of last sensitivity function

  • dist_hard_max ‚Äì cutoff for cutoff function.

  • n_interaction_layers ‚Äì number of interaction blocks

  • n_atom_layers ‚Äì number of atom layers per interaction block.

  • possible_species ‚Äì list of species values in database, including 0 for none. For example, HCNO would be [0 1 6 7 8]

  • n_input_features ‚Äì number of input features to the model

  • sensitivity_type ‚Äì str or callable, type of sensitivity, default of ‚Äòinverse‚Äô is what is in hip-nn original paper.

  • resnet ‚Äì bool or int, if int, size of internal resnet width

  • activation ‚Äì activation function or subclass of nn.module.

Note: only one of possible_species or n_input_features is needed. If both are supplied, they must be consistent with each other.
